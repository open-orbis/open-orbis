## 2. Direct Competitors (Individual Professional Identity)

### 2.1 LinkedIn
- **What it is:** The dominant professional network (1.2B+ users)
- **Overlap with Orbis:** Professional profiles, recruiter search, networking
- **Key difference:** Walled garden, feed-first, not machine-queryable, no data portability, no graph visualization
- **Pricing:** Free (basic), Premium Career ($30/mo), Recruiter Lite ($170/mo), Recruiter Corporate ($835+/mo)
- **Orbis advantage:** Open graph via MCP, full data portability (JSON/JSON-LD/PDF), E2E encryption, no algorithm-driven feed noise
- **Orbis risk:** LinkedIn's 1.2B-user network effect is the single biggest adoption barrier

### 2.2 Read.cv
- **What it is:** A polished, minimal professional profile platform — "one link" for your professional identity
- **Overlap with Orbis:** Professional presence, shareable link, portfolio
- **Key difference:** Static profile (not a graph), no AI/agent interoperability, no recruiter tools, no structured data export
- **Pricing:** Free
- **Orbis advantage:** Graph structure, MCP queryability, semantic search, 3D visualization, recruiter tier
- **Orbis risk:** Read.cv is simpler and already has traction among designers and engineers

### 2.3 Polywork
- **What it is:** A multi-faceted professional profile for "multi-hyphenate" careers
- **Overlap with Orbis:** Rich professional profiles, shareable links, collaboration highlights
- **Key difference:** No graph structure, no AI-native features, no MCP, declining traction
- **Pricing:** Free (basic), Pro ($8/mo)
- **Orbis advantage:** Graph-native, AI-queryable, recruiter marketplace
- **Note:** Polywork has struggled with growth; the multi-hyphenate positioning didn't scale

### 2.4 Peerlist
- **What it is:** A "proof-of-work" professional platform for developers and designers
- **Overlap with Orbis:** Professional profiles, project showcasing, peer connections
- **Key difference:** Focused on social proof (GitHub contributions, projects), not structured data or AI interop
- **Pricing:** Free (basic), Pro ($5/mo)
- **Orbis advantage:** Deeper data structure, MCP tools, recruiter analytics, semantic search

## 3. Direct Competitors (Structured Resume / CV Tools)

### 3.1 JSON Resume (Open Source Standard)
- **What it is:** An open-source JSON schema for resumes — store your CV as structured data
- **Overlap with Orbis:** Structured, portable, machine-readable CV data
- **Key difference:** Schema only (no platform, no visualization, no hosting, no recruiter tools, no AI interop)
- **Pricing:** Free (open-source)
- **Orbis advantage:** Full platform (hosting, 3D viz, MCP, recruiter tier, encryption). JSON Resume is a standard; Orbis is a product
- **Orbis opportunity:** Support JSON Resume import/export for interoperability

### 3.2 Reactive Resume
- **What it is:** The most popular open-source resume builder (30K+ GitHub stars). Uses JSON Resume schema under the hood
- **Overlap with Orbis:** Structured CV data, privacy-focused, self-hostable, JSON export
- **Key difference:** Outputs traditional PDF resumes (not a living graph), no network effects, no recruiter marketplace, no AI/MCP interop
- **Pricing:** Free (open-source, self-hosted)
- **Orbis advantage:** Living graph vs. static document, MCP tools, recruiter tier, 3D visualization, orb-to-orb connections
- **Orbis risk:** Reactive Resume has strong developer loyalty and 30K+ stars; it's the incumbent in open-source resume tools

### 3.3 Standard Resume / FlowCV / Novoresume / Zety
- **What it is:** Commercial resume builders (template-based, PDF output)
- **Overlap with Orbis:** CV creation
- **Key difference:** Template-focused, static PDF output, no structured data, no AI interop
- **Pricing:** Free (basic), $10–25/mo (premium templates)
- **Orbis advantage:** Orbis eliminates the need for templates entirely — the graph IS the resume

## 4. Direct Competitors (Recruiter-side Talent Intelligence)

### 4.1 Eightfold AI
- **What it is:** Enterprise talent intelligence platform with a deep-learning skills graph mapping 1.5B+ talent profiles
- **Overlap with Orbis:** Skills graph, talent search, structured professional data, AI-native
- **Key difference:** Enterprise-only (not individual-facing), extremely expensive, closed data, no individual user control
- **Pricing:** Enterprise contracts ($100K–$1M+/year)
- **Orbis advantage:** Bottom-up (individuals own their data), open via MCP, privacy-first, vastly cheaper
- **Orbis risk:** Eightfold has massive funding ($2.1B valuation) and deep enterprise relationships

### 4.2 Beamery
- **What it is:** Talent lifecycle platform with "Talent Graph" technology and Skills Ontology
- **Overlap with Orbis:** Talent graph, skills mapping, candidate profiles, AI-powered matching
- **Key difference:** Enterprise CRM (not individual-facing), top-down data aggregation, no individual data ownership
- **Pricing:** Enterprise contracts
- **Orbis advantage:** Individual-first, data portability, MCP interoperability, transparent and user-controlled
- **Complementary opportunity:** Beamery could consume Orbis orbs via MCP as a data source

### 4.3 hireEZ
- **What it is:** AI sourcing platform aggregating 750M+ profiles from 45+ platforms, now with agentic AI (EZ Agent)
- **Overlap with Orbis:** Structured professional data, AI-powered search, recruiter tools
- **Key difference:** Scrapes public data (individuals don't control it), recruiter-only, no individual product
- **Pricing:** Custom enterprise pricing
- **Orbis advantage:** Consent-based (individuals opt in), richer structured data (graph vs. scraped profiles), MCP-native
- **Orbis risk:** hireEZ already has 750M+ profiles; Orbis starts from zero

### 4.4 SeekOut
- **What it is:** Talent search engine with diversity analytics and internal talent marketplace
- **Overlap with Orbis:** Talent search, skills mapping
- **Key difference:** Enterprise-only, scrapes public data
- **Pricing:** Enterprise contracts
- **Orbis advantage:** Same as vs. hireEZ — consent-based, individual-owned, MCP-native

## 5. Adjacent / Emerging Competitors

### 5.1 Apt
- **What it is:** AI-powered career development platform combining career counseling, job board, and mentorship
- **Overlap with Orbis:** AI-native career tool, professional identity
- **Key difference:** Focused on career guidance (not data structure), no graph, no MCP, no recruiter marketplace
- **Orbis advantage:** Structured data layer, recruiter tier, agent interoperability

### 5.2 Contra
- **What it is:** Platform for independent/freelance professionals with portfolio showcasing and project-based connections
- **Overlap with Orbis:** Professional profiles, project showcasing
- **Key difference:** Freelance-focused, marketplace model, no graph structure
- **Orbis advantage:** Graph-native, broader applicability (not just freelancers)

### 5.3 Lunchclub
- **What it is:** AI-powered 1-on-1 professional networking via curated introductions
- **Overlap with Orbis:** AI-powered professional connections
- **Key difference:** Networking-only (no persistent identity/data layer), no recruiter tools
- **Orbis advantage:** Persistent professional graph, data portability, recruiter tier

## 8. Strategic Implications for Orbis

1. **The real competition is inertia, not products.** LinkedIn's 1.2B users and the habit of "just upload a PDF" are harder to displace than any specific competitor.

2. **Position against LinkedIn's weaknesses**, not its strengths. Don't try to be a social network. Win on: data ownership, AI interoperability, privacy, and structured queryability.

3. **Embrace open-source adjacents.** Support JSON Resume import, integrate with Reactive Resume exports, and position Orbis as the "hosted, AI-native layer" on top of open standards.

4. **The recruiter-side competitors validate the market.** Eightfold ($2.1B), Beamery, hireEZ all prove that structured talent graphs have massive enterprise value. Orbis's innovation is making the individual the data owner.

5. **MCP is the moat — for now.** No competitor has MCP-native professional identity. This window is 12–18 months before incumbents adopt it. Move fast.

6. **Potential partnerships > competition** with Beamery, hireEZ, and ATS platforms. Orbis orbs consumed via MCP by enterprise talent tools is a B2B revenue channel that doesn't require beating LinkedIn at networking.
